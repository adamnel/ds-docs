A checkbox allows the user to select one or more options from a set of choices. Unlike radio buttons, checkboxes allow for multiple selections, meaning the user can choose more than one option at a time.

## Usage

<imagePlaceholder imageId="cv-checkbox_hero" />

### When to use

Checkboxes should be used when multiple selections are allowed from a set of options. If the user can only choose one option from a list, use **radio buttons** instead.

**Variants**

| Variant | Purpose |
| --- | --- |
| **Default** | Allow users to select multiple options independently. |

**Related:** [Form field](https://teradata.github.io/covalent/docs/components/?path=/docs/components-form-field--overview) wraps a checkbox with its label.

### Recommendations

<imagePlaceholder imageId="cv-checkbox_do_labels" />

<doDont
  doBody={<>
    Make sure each checkbox has a **clear and concise label**. Provide clear labels and group checkboxes logically. Use checkboxes for agreeing to terms, selecting multiple items, or confirming options. Ensure targets are large enough on all screen sizes. When helpful, include a **select all** control.
  </>}
  dontBody={<>
    Use checkboxes for **mutually exclusive** choices—use **radio buttons** instead. Overuse checkboxes for secondary options, leave state ambiguous, or rely on the control alone without labels and context.
  </>}
/>

### Content guidelines

The checkbox helps users make binary choices. Use a direct, supportive voice; keep labels to **3–5 words** where possible; helper text up to ~50 characters.

## Accessibility

### Overview

Checkboxes support multiple selections and require clear labeling, visible focus, and error handling that does not rely on color alone.

### Keyboard

- **Tab** moves focus to the checkbox.
- **Space** toggles the checkbox.

<imagePlaceholder imageId="cv-checkbox_accessibility1" />

### Focus order

- Use **`tabindex="0"`** on active checkboxes; exclude disabled controls from the tab order.
- Pair focus rings with selection state so sighted users can follow focus.

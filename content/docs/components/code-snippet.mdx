Portion of code displayed within an interface, often for demonstration, configuration, or explanation.

## Usage

Use code snippets when users must **copy**, **inspect**, or **compare** short pieces of code or structured text. Keep snippets **scoped** (only what the task needs), **readable** (line length and wrapping), and **accessible** (syntax highlighting does not replace structure—expose plain text for assistive tech).

## Accessibility

- Ensure code is available as **text**, not only as an image.
- If you provide **copy**, expose **name** and **state** (e.g., “Copied”) to screen readers.
- Avoid relying on **color** alone for syntax—structure and copy affordances still matter for low-vision users.

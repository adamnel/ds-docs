## Web components

### Properties

| Property | Description |
| --- | --- |
| `covalentIcon` | Use a Covalent icon |
| `filledIcon` | Use a filled icon variant |
| `icon` | Icon name to display |
| `scale` | Text scale. One of: `body1`, `body2`, `caption`, `headline1`, `headline2`, `headline3`, `headline4`, `headline5`, `headline6`, `subtitle1`, `subtitle2`, `overline`, `button` |
| `state` | Visual state. One of: `caution`, `negative`, `positive`, `primary` |

```html
<cv-icon-lockup scale="body1" icon="houseboat">
  Lorem ipsum dolor
</cv-icon-lockup>
```

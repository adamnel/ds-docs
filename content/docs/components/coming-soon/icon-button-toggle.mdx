## Web components

### Properties

| Property | Description |
| --- | --- |
| `on` | Whether the toggle is on |
| `disabled` | Disables the toggle |
| `onIcon` | Icon displayed when toggled on |
| `offIcon` | Icon displayed when toggled off |
| `aria-label` | Accessible label |
| `ariaLabelOn` | Accessible label when on |
| `ariaLabelOff` | Accessible label when off |

```html
<cv-icon-button-toggle onIcon="flashlight_on" offIcon="flashlight_off"></cv-icon-button-toggle>
```

### Events

| Event name | Detail | Description |
| --- | --- | --- |
| `icon-button-toggle-change` | `{isOn: boolean}` | Indicates the button has been toggled. `isOn` indicates the `on` value of the toggle button. |

## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `selectable` | `boolean` | **Required.** Whether chips in the set are selectable. |
| `multi` | `boolean` | Whether multiple chips can be selected at once. |
| `label` | `string` | Accessible label for the chip set. |
| `activeIndex` | `number` | Index of the currently active chip. |

### Example

```html
<cv-chip-set selectable activeIndex="0" label="chip-set">
  <cv-chip label="Chip" filter state="secondary"> </cv-chip>
  <cv-chip label="Chip" filter state="secondary"> </cv-chip>
  <cv-chip label="Chip" filter state="secondary"> </cv-chip>
</cv-chip-set>
```

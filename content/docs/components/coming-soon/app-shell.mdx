## Web components

### Properties

| Property | Description |
| --- | --- |
| `appName` | The application name displayed in the shell. |
| `sectionName` | The name of the current section. |
| `helpOpen` | Whether the help panel is open. |
| `drawerOpen` | Whether the navigation drawer is open. |
| `contained` | Whether the shell is contained within a parent element. |
| `forcedOpen` | Whether the drawer is forced open. |

### Slots

| Slot | Description |
| --- | --- |
| `default` | Main content area. |
| `logo` | Logo displayed in the app shell header. |
| `navigation` | Navigation list content for the drawer. |
| `help` | Help panel content. |
| `section-action` | Action items for the section header. |
| `user-menu` | User menu content. |

## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `indeterminate` | `boolean` | Whether the progress indicator is indeterminate (no specific progress value). |
| `progress` | `number` | Current progress value (0 to 1). |
| `density` | `number` | Density scale for the component size. |
| `closed` | `boolean` | Whether the progress indicator is hidden. |

### Example

```html
<cv-circular-progress indeterminate progress="0" density="0"></cv-circular-progress>
```

### Methods

| Method | Description |
| --- | --- |
| `open() => void` | Sets `closed` to `false`, showing the progress indicator. |
| `close() => void` | Sets `closed` to `true`, hiding the progress indicator. |

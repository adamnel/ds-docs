## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `icon` | `string` | Icon to display for the item. |
| `isOpen` | `boolean` | Whether the item is expanded. |
| `indentLevel` | `number` | The nesting level of the item. |
| `indentMultiple` | `number` | Multiplier for indent spacing. |

### Slots

| Slot | Description |
| --- | --- |
| `label` | Primary text label for the item. |
| `extraContent` | Secondary text or additional content. |
| `nest` | Nested child items. |

```html
<cv-tree-list-item icon="description">
  <div slot="label">Tree list item</div>
  <div slot="extraContent">Secondary text</div>
</cv-tree-list-item>
```

## Web components

### Properties

| Property | Description |
| --- | --- |
| `selected` | Whether the switch is on |
| `disabled` | Disables the switch |
| `name` | Form field name |
| `value` | Form field value |

```html
<cv-switch value="on"> </cv-switch>
```

## Web components

### Properties

| Property | Description |
| --- | --- |
| `activeIndex` | Index of the active tab |

```html
<cv-tab-bar activeIndex="0">
  <cv-tab label="Tab label"> </cv-tab>
  <cv-tab label="Tab label"> </cv-tab>
  <cv-tab label="Tab label"> </cv-tab>
</cv-tab-bar>
```

### Methods

| Method signature | Description |
| --- | --- |
| `scrollIndexIntoView(index: number) => void` | For long, scrollable `cv-tab-bar`s, scrolls the tab at the given index into view. |

### Events

| Event name | Target | Detail | Description |
| --- | --- | --- | --- |
| `MDCTabBar:activated` | `cv-tab-bar` | `{index: number}` | Emitted when a tab selection has been made. |

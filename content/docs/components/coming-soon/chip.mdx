## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `label` | `string` | **Required.** The text label for the chip. |
| `state` | `"secondary" \| "negative" \| "positive" \| "caution"` | Visual state variant. |
| `icon` | `string` | Leading icon name. |
| `filter` | `boolean` | Whether the chip is a filter chip (shows checkmark when selected). |
| `trailingIcon` | `string` | Trailing icon name. |
| `avatar` | `boolean` | Whether to show an avatar. |
| `disabled` | `boolean` | Whether the chip is disabled. |

### Example

```html
<cv-chip label="Chip" filter state="secondary"> </cv-chip>
```

## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `persistent` | `boolean` | Whether the tooltip stays visible without hover. |
| `rich` | `boolean` | Whether to use the rich tooltip variant. |
| `richTitle` | `string` | Title text for rich tooltips. |
| `hideDelay` | `number` | Delay in milliseconds before the tooltip hides. |
| `showDelay` | `number` | Delay in milliseconds before the tooltip shows. |
| `actionItems` | `slot` | Slot for action items in rich tooltips. |

```html
<cv-tooltip hidedelay="5000" open>
  Im a tooltip
</cv-tooltip>
```

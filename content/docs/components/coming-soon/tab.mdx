## Web components

### Properties

| Property | Description |
| --- | --- |
| `label` | Tab label text |
| `icon` | Icon name |
| `indicatorIcon` | Indicator icon |
| `hasImageIcon` | Whether the icon is an image |
| `isFadingIndicator` | Uses a fading indicator |
| `minWidth` | Uses minimum width |
| `isMinWidthIndicator` | Uses minimum width indicator |
| `stacked` | Stacks icon above label |
| `active` | Whether the tab is active |

```html
<cv-tab label="Tab label"> </cv-tab>
```

### Methods

| Method signature | Description |
| --- | --- |
| `activate() => void` | Activates the indicator. |
| `deactivate() => void` | Deactivates the indicator. |
| `computeDimensions() => MDCTabDimensions` | Computes the dimensions of the content. See [Tab types](https://github.com/material-components/material-components-web/blob/master/packages/mdc-tab/types.ts#L29) for definition. |
| `computeIndicatorClientRect() => ClientRect` | Computes the dimensions of the indicator. |

### Events

| Event name | Target | Detail | Description |
| --- | --- | --- | --- |
| `interacted` | `cv-tab` | `{tabId: string}` | Emitted when the Tab is interacted with, regardless of its active state. Used by parent components to know which Tab to activate. |

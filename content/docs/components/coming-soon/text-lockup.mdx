## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `subText` | `string` | Secondary text displayed below the main text. |
| `scale` | `string` | Size of the text lockup. One of: `large`, `small`. |
| `icon` | `string` | Icon to display. |
| `state` | `string` | Visual state. One of: `active`, `positive`, `negative`, `caution`. |
| `trailingSubText` | `string` | Secondary text displayed after the main text. |

```html
<cv-text-lockup subText="environment Id" scale="small">
  BXCVP-8523411
</cv-text-lockup>
```

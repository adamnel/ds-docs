## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `cardTitle` | `string` | The title displayed in the card header. |
| `outlined` | `boolean` | Whether to use an outlined card style. |
| `actionBar` | `boolean` | Whether to show the action bar. |
| `interactive` | `boolean` | Whether the card is interactive (clickable). |

### Slots

| Slot | Description |
| --- | --- |
| `default` | Main card content. |
| `card-actions` | Action buttons displayed in the card action bar. |

### Example

```html
<cv-card cardTitle="Card title" actionBar>
  <div slot="card-actions">
    <cv-button label="Retry" outlined> </cv-button>
    <cv-icon-button icon="houseboat"> </cv-icon-button>
  </div>
  <cv-empty-state
    headline="You don't have access to this area"
    subtitle="You don't have permission to view this application."
    icon="lock">
    <div slot="button"></div>
    <div slot="buttons"></div>
  </cv-empty-state>
</cv-card>
```

## Web components

The `cv-typography` component provides a set of predefined text scales for consistent typographic styling.

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `scale` | `string` | The typographic scale to apply. One of: `headline1`, `headline2`, `headline3`, `headline4`, `headline5`, `headline6`, `subtitle1`, `subtitle2`, `body1`, `body2`, `caption`, `button`, `overline`. |

### Available scales

* Headline 1
* Headline 2
* Headline 3
* Headline 4
* Headline 5
* Body 1
* Body 2

```html
<cv-typography scale="headline1">
  The quick brown fox jumps over the lazy dog
</cv-typography>
```

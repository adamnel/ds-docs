## Web components

### Properties

| Property | Description |
| --- | --- |
| `icon` | Icon to display |
| `aria-label` | Accessible label |
| `aria-haspopup` | Indicates popup behavior |
| `disabled` | Disables the button |

```html
<cv-icon-button icon="houseboat"> </cv-icon-button>
```

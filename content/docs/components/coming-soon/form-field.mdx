## Web components

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `label` | `string` | **Required.** The label displayed next to the form control. |
| `alignEnd` | `boolean` | Whether to align the label at the end. |
| `spaceBetween` | `boolean` | Whether to add space between the label and the control. |
| `nowrap` | `boolean` | Whether to prevent the label from wrapping. |

### Example

```html
<cv-formfield label="Form field">
  <cv-checkbox value=""> </cv-checkbox>
</cv-formfield>
```

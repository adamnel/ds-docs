## Web components

### Properties

| Property | Description |
| --- | --- |
| `buffer` | Buffer value |
| `progress` | Progress value |
| `indeterminate` | Enables indeterminate mode |
| `reverse` | Reverses the direction |
| `closed` | Whether the progress bar is closed |

```html
<cv-linear-progress indeterminate> </cv-linear-progress>
```

### Methods

| Method signature | Description |
| --- | --- |
| `open() => void` | Sets `CovalentLinearProgress.closed` to `false` |
| `close() => void` | Sets `CovalentLinearProgress.closed` to `true` |

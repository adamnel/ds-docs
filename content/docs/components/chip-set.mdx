Compact visual element representing an attribute, action, or choice. A **chip set** groups multiple [Chip](https://teradata.github.io/covalent/docs/components/?path=/docs/components-chips--overview) controls for selection, filtering, or actions.

## Usage

Design documentation for chip set continues to align with **Material chips** patterns. Review the current Figma documentation and Storybook examples before shipping new layouts.

### When to use

Use chip sets when users pick **one or many** options from a compact list—filters, facets, or removable tokens. Prefer **radio** or **select** when the list is long or requires heavy scanning.

## Web components

### Example

```
<cv-chip-set selectable activeIndex="0" label="chip-set">
  <cv-chip label="Chip" filter state="secondary"></cv-chip>
  <cv-chip label="Chip" filter state="secondary"></cv-chip>
  <cv-chip label="Chip" filter state="secondary"></cv-chip>
</cv-chip-set>
```

### Common props

| Name | Role |
| --- | --- |
| `selectable` | Enables selection behavior |
| `multi` | Allows multiple selection where supported |
| `label` | Accessible label for the set |
| `activeIndex` | Controlled active chip index |

Our design system is here to help you build with consistency, efficiency, and quality. Whether you are a designer crafting intuitive experiences or a developer implementing scalable solutions, these guides provide the key resources you need.

- **Designers** — Set up your workspace, access design files, and apply system components efficiently. See the [Covalent repository](https://github.com/Teradata/covalent) for design assets and contribution paths.
- **Developers** — Install Covalent packages, use web components or framework bindings, and follow implementation best practices. Start with the [published Storybook](https://teradata.github.io/covalent/docs/components/) and the repo README.

Use the sidebar to explore **Foundations**, **Components**, **Patterns**, **Tokens**, **Content**, and **Feedback** as you build.

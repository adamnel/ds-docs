# Toolbar / Accessibility

Provide wayfinding, context and actions for the current page

# 

Overview

Toolbars are designed to be accessible to everyone, enabling all users, regardless of ability, to understand the context of a page and take action there effectively.

# 

Keyboard interactions

To support accessibility through keyboard navigation, toolbars include the following interactions:

*   **Tab key:** Focuses the buttons on the toolbar.
    
*   **Enter/spacebar:** Selects the button currently focused.
    

## 

Tab index focus

Ensures that toolbars are included in the sequential keyboard navigation, allowing users to reach and interact with the toolbar buttons using the Tab key.

## 

Hit area state

A visible focus indicator highlights the toolbar buttons when they gain keyboard focus, ensuring that users can easily identify which element is active.

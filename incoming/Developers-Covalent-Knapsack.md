# 

Feature requests or issues

Did you find an issue or have a feature request with one of our existing components? First, check through [our GitHub Issues](https://github.com/Teradata/covalent/issues). If you don’t see it listed, go ahead and create an issue. 

Also, if you do not find what you’re looking for, reach out via Slack in our [#design-system](https://teradata.slack.com/archives/C02GW1QHBMZ) channel. We can help guide you in the right direction depending on your needs.

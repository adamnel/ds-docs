# Cards / Accessibility

Cards serve as shortcuts to tasks that we want people to complete first. It's an opportunity for us to help them quickly unlock the value of their Teradata feature or product.

## 

Overview

Accessibility is paramount in ensuring that Get Started Cards are inclusive and effective for all users, including those with disabilities. These cards are designed to help users quickly engage with key features and tasks during initial product interaction or setup. To support accessibility, the design and implementation of these cards adhere to WCAG 2.1 standards, focusing on navigability, clarity, and ease of interaction through both traditional and assistive technologies

## 

Keyboard interactions

### 

Key Controls

*   **Tab Key:** Advances the focus from one interactive element to the next within the Get Started Cards.
    
*   **Enter/Spacebar:** Activates the focused element or action, such as expanding a card detail or initiating an action linked to the card.
    

### 

Focus Management

*   **tabindex="0":** Assigns focusability to interactive elements within the Get Started Cards, making them part of the sequential keyboard navigation.
    
*   **Visual Focus Indicators:** Provides clear and noticeable indicators when elements are focused. This helps in identifying the current interactive element.
    
*   **Exclusion of Non-Interactive Elements:** Ensures that elements which do not require user interaction, like static text, do not receive focus to streamline navigation and avoid confusion.

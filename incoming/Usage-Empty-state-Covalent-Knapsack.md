# 

When to use

Provides feedback and guidance in the event that content can’t be shown, like when no data is available.

**Variants**

**Purpose**

Empty cards

Provides feedback and guidance in the event that content can’t be shown on a card.

Empty pages

Provides feedback and guidance in the event that content can’t be shown on a page.

Empty autocomplete

Provides feedback and guidance in the event that no content is available in an autocomplete dropdown.

# 

Recommendations

  

##### Do

*   **Do** make it clear if this is an expected situation or an error.
    
*   **Do** provide a next step for the user to take to resolve the situation if possible.
    

##### Don't

*   **Don't** steal visual focus from the rest of the page.

# Radio / Style

A radio button allows the user to select a single option from a set of mutually exclusive choices. It is typically used when only one option can be selected from a predefined set.

# 

Content guidelines

Radio buttons allow users to select one option from a set. Clear labels and hierarchy reduce confusion.

*   **Categorization**: Use for mutually exclusive options only.
    
*   **Voice & tone**: Friendly and concise.
    
*   **Word limit**: 2-4 words for button labels. Group label or helper text up to 50 characters.
    

##### Do

**Example Label**

"Monthly Plan"

##### Don't

**Example Label**

**"**Choose this option if you want a monthly plan."

# 

Name your file

Change the content and state of the status card in your Figma file as you progress through the design process. If you're using the new feature template, the status card will be on the Overview page.

Each title is made up of one item from each of the following categories:

*   Design status
    
    *   Completed
        
    *   In dev
        
    *   Review
        
    *   In progress
        
    *   Next up
        
    *   Not started
        
    *   Blocked
        
    *   Legacy
        
    *   Archived
        
*   Ticket number
    
    *   For example VEDITOR-294
        
*   Title
    
    *   For example Editor enhanced layout

# 

Update the status as you progress

The design status names are chosen so that if you sort the Figma 💟 **Features** project reverse alphabetically, the ones that need attention the most (in progress and review) will rise to the top. This lets us all view the project and see quickly what everyone is working on and stay abreast of changes.

To pick up a draggable item, press the space bar. While dragging, use the arrow keys to move the item. Press space again to drop the item in its new position, or press escape to cancel.

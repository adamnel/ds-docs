# Radio / Usage

A radio button allows the user to select a single option from a set of mutually exclusive choices. It is typically used when only one option can be selected from a predefined set.

# 

When to use

Use radio buttons when you need the user to choose one option from a set of predefined options, such as selecting a payment method or choosing a shipping speed. If there are more than two options, or the choices are less mutually exclusive, consider using [checkboxes](#) or a [drop-down menu.](#)

##### Do

Use radio buttons for mutually exclusive choices (e.g., "Male" / "Female," "Credit Card" / "PayPal").

##### Don't

Use radio buttons for selections where the user needs to choose more than one option—use **checkboxes** for multiple selections

![cv-radio_do-heading](https://design-src.knapsack.cloud/figma/covalent/KvXVAmYdCVAS7hzkOa73Em/2261921067357318529/images/14705-6140.png?Expires=1768218598&Key-Pair-Id=K3CKBA7ZQV44SC&Signature=s-BbwBvE~F1dd9--lN4M8O5bzrVybTiM30o8sn~l-h88rjutA9uhxhwUlIGM7frglsRyhvmRaTgck-ySY2rwVt8ZrZk9omEky2SfWyE13PdiMEEthLoLCH43xyy0qk6E3uiJTnKCiwvhVcDWh8fmIH7v4yRqbhRq~TqOZnwav6~a9C8Y1TGK4~550Nd1y2z7dkMh2A6TswveWWVjOGv4OpVbe-uM0GxiqoKf7hlZUmD-82ZEx49IC3~rKGmUuATaqpyrDMGN4fBTEgJxmRZlnR92mL7wU0BT6d4AB~WrQx7aSzPGA6lsvALy66LfvRsh66T-qjk4XCEwXOUpEWLTZw__)

##### Do

Label each radio button clearly and provide a grouping heading.   
  

##### Do

*   Ensure radio buttons are clearly labeled with descriptive options. 
    
*   Ensure that the user cannot select multiple radio buttons in a set. 
    
*   Group radio buttons together. 
    
*   Provide a default selection if appropriate (e.g., defaulting to the most common choice). 
    
*   Make sure radio buttons are appropriately spaced to avoid accidental selection
    

##### Don't

*   Make radio buttons too small to be easily clickable, especially on mobile.

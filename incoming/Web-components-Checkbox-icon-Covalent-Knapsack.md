Fired when the user modifies the checkbox `checked` state from an input device interaction. Note that, like [native](https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement/change_event) `<input>`, the `change` event is _not_ fired when the `checked` property is set from JavaScript.

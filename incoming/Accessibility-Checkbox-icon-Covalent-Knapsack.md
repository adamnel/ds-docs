# Checkbox icon / Accessibility

Checkbox styled with an icon as its visual representation.

# 

Overview

Checkboxes support multiple selections and require clear labeling, focus states, and error handling to guide user actions effectively.

# 

Keyboard interactions

Checkbox icons should be fully operable via keyboard:

*   **Tab key**: Focuses the checkbox icon.
    

*   **Spacebar**: Toggles the checkbox icon on or off.
    
*   **Arrows**: Moves the selection.

Tab index focus

*   **tabindex="0"** for active checkboxes.
    
*   Provide clear focus and selection indicators.
    
*   Exclude disabled checkboxes from the tab order.
